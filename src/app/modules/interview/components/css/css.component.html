Purpose of meta|viewport and doctype?

<meta> tag provides metadata about the HTML document. It will not displayed on the page, but will be machine parsable.
      It is used to specify page description, keywords, author of the document last modified and other metadata.
      It can be used by browsers (how to display content or reload page), and search engine keywords.

<meta name="description/keywords/author/viewport" />


Viewport:- The viewport is the users visible area of a webpage. It varies with device.

<meta name="viewport" content="width=device-width" initial-scale=1.0 />

viewport elements gives the browser instructions on how to control the pages dimension and scaling

Doctype:- HTML5 requires a Doctype to be specified to ensure that the browser render the page is standard mode.

<!DOCTYPE html >
